Unit Nedir : Bir obje'nin bir davranışıdır. 

Unit Test Nedir : unit test ise bu unitlerin yani davranışların herbirinin ayrı ayrı test edilmesidir.
davranışların arasındaki bağa bakılmadan herbiri ayrı ayrı test edilir.

Michael Feathers'e göre test kuralları :
*Veritabanıyla konuşuyor
*Ağ üzerinden iletişim kurar
*Dosya sistemine dokunuyor
*Diğer birim testlerinizle aynı anda çalışamaz
*Çalıştırmak için ortamınıza özel şeyler (yapılandırma dosyalarını düzenlemek gibi) yapmanız gerekir.
bunları yapıyorsak bu testler unit test sayılmaz bu testlerde yapılması gerekiyor fakat bunlar unit test sayılmaz.
